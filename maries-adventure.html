✦ HOW TO PLAY ✦

🗺 Select a Realm from the Kingdom map to start.

📖 Read the story — Marie faces a new adventure each level!

🔢 Solve 3 math puzzles to unlock each realm.

❤️ You have 3 hearts per puzzle — wrong answers cost one!

⭐ Earn 1–3 stars per realm based on accuracy.

💡 Use the Hint button if you're stuck — it's free!

🏆 Beat all 10 realms to save the Magical Kingdom!

Math Topics: Addition → Subtraction → Multiplication → Division → Fractions → Patterns → Geometry → Algebra → Mixed Challenge → Final Boss
